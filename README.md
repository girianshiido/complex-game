# Astra Complex

Un jeu pédagogique autonome pour s’entraîner au calcul avec les nombres complexes.

## Fonctionnalités

- quatre entraînements ciblés et un défi mixte ;
- difficulté adaptative sur trois niveaux ;
- explication détaillée après chaque réponse ;
- bilan de fin de série et record enregistré localement ;
- thèmes clair et sombre ;
- navigation au clavier et mise en page adaptée aux mobiles ;
- aucune inscription, aucun serveur et aucune donnée transmise.

## Lancer le jeu

Ouvrir `index.html` dans un navigateur ou servir le dossier avec un serveur web local.

## Publication

Le projet est conçu pour être publié directement avec GitHub Pages depuis la branche principale.
